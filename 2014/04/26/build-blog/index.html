该博客搭建笔记

使用Hexo搭建博客，markdown写博客，生成的静态页面托管在Github，图床为七牛和阿里云OSS

1.搭建环境

  • 安装git，node.js，hexo(git bash中执行$ npm install -g hexo)
  • 新建文件夹blog，在其中执行git bash命令：$ hexo init—>$ hexo generate—>$ hexo server 访问localhost:4000本地博客
  • 在github中新建一个repository:zlvip007.github.io

  • 编辑_config.yml如下：
    deploy:
    type: github
    repository: https://github.com/zlvip007/zvip007.github.io.git
    branch: master

  • $ hexo generate—>$ hexo deploy 完成部署，访问zlvip007.github.io

2.写博客

  • $ hexo new "new post" 生成blog/source/_post/new_post.md,用markdown编写后
  • $ hexo deploy 发布
  • 完毕，访问 zlvip007.github.io

Hexo官方文档摘要

Writing

$ hexo new [layout] "title"
[layout]
post(Default) source/_posts
draft source/_draft
page source

<!--more--> 使文章显示前半部分

Generating

$ hexo g generate static files
deploy after generating in two ways:
$ hexo g -d
$ hexo d -g

Server

$ hexo s start server
server会自动更新file，但public中的不会，需$ hexo g
配置文件更新需重启

Deployment

$ hexo d deploy
$ rm -rf .deploy remove .deploy folder
在source中新建CNAME,放置自己的域名

Configuration

subtitle: connecting dots...
